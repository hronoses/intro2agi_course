# Матеріали до курсу "Вступ до штучного загального інтелекту"
Цей репозиторій містить матеріали до курсу "Вступ до штучного загального інтелекту". Тут ви знайдете інструкції, код та інші ресурси для виконання практичних завдань.



git clone https://github.com/hronoses/intro2agi_course
cd intro2agi_course
git checkout -b your_branch_name


pip install -r requirements.txt

Після виконання цих команд ви створите нову гілку для роботи над практикою. Не забудьте замінити `your_branch_name` на відповідну назву вашої гілки.
